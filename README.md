# Duitse Beurs Trainer (YarnZoo Edition)

Een interactieve web-app om Duitse verkoopgesprekken te oefenen, specifiek voor de haak- en garenindustrie. Ontwikkeld voor YarnZoo om te trainen voor Duitse vakbeurzen.

## Features
- **Scenario Training**: Oefen specifieke situaties zoals begroetingen, productuitleg (Amigurumi, Moza√Øek) en B2B onderhandelingen.
- **Snel Oefenen (Drill Mode)**: Kies 3, 5 of 10 willekeurige vragen voor een snelle training.
- **Spraakherkenning**: Oefen je uitspraak en krijg direct feedback.
- **Gamification**: Verdien XP punten en houd je voortgang bij (lokaal opgeslagen).
- **YarnZoo Branding**: Volledig in de stijl van YarnZoo (kleuren, lettertypes, logo).

## Technologie
- React
- Tailwind CSS (v4)
- Web Speech API (Text-to-Speech & Speech-to-Text)

## Lokaal Draaien

1. Clone de repository
2. Installeren: `npm install`
3. Starten: `npm run dev`
